## 프로젝트 개요

**바야다(BAYADA)**는 요양보호사와 돌봄이 필요한 시니어/보호자를 연결하는 **양면 매칭 플랫폼**입니다.

### 핵심 컨셉

```
요양보호사(공급) ←→ 바야다 플랫폼 ←→ 보호자/환자(수요)
                        ↕
              요양기관(B2B) + 정부/지자체(B2G)
```

### 비전
> "모든 어르신이 존엄한 돌봄을 받고, 모든 요양보호사가 정당한 대우를 받는 세상"

### 미션
> 기술과 데이터를 통해 시니어 돌봄의 품질, 접근성, 투명성을 혁신한다

### 슬로건
> **"돌봄의 새로운 기준, 바야다"**

| 맘시터 성공 전략 | 바야다 적용 |

| 프리미엄화 (프로케어/하이시터) | **관리형 돌봄** (전담 매니저 + VIP 서비스) |

| 10단계 인증 | **7단계 인증** (자격증+경력+건강+범죄경력+면접+교육+보증) |
| 째깍악어 O2O (째깍섬) | **오프라인 돌봄 체험관** (장기 검토) |
