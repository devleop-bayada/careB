# Part 1. 글로벌 시장 개요

## 1.1 글로벌 보육 시장 규모

| 지표 | 수치 |
|------|------|
| 2024년 글로벌 보육 서비스 시장 | **USD 3,430억** (Grand View Research) |
| 2025년 글로벌 보육 시장 | USD 2,451억 (Towards Healthcare) |
| 2030년 예상 규모 | USD 3,964억~4,423억 |
| 2035년 예상 규모 | USD 4,275억 |
| CAGR (2025-2030) | **4.3%~5.72%** |

**미국 시장:**
- 2024년: USD 651.5억 → 2033년: USD 1,098.8억 (CAGR 6.02%)

**베이비시팅 앱 시장:**
- 2023년: USD 24억 → 2033년: USD 65억 (CAGR 10.4%)
- 아시아 태평양 지역이 가장 빠른 성장률 (**CAGR 15%**)

**AI in Childcare 시장:**
- 2024년: USD 47억 → 2034년: USD 352억 (**CAGR 22.4%**)

**보육 소프트웨어 시장:**
- 2025년: USD 6.5억 → 2035년: USD 12.4억 (CAGR 6.7%)

## 1.2 글로벌 시장 성장 동인

| 동력 | 세부 내용 |
|------|-----------|
| 맞벌이 가구 증가 | 특히 개발도상국에서 여성 경제활동 참가율 상승 |
| 핵가족화 | 전통적 대가족 지원 체계 약화 |
| 원격근무 확산 | 유연한 돌봄 수요 증가 |
| 정부 보육 지원 확대 | 각국 보육 보조금 및 세제 혜택 확대 |
| 기업 복지 트렌드 | 33%의 고용주가 백업케어 제공 (2025년 SHRM 조사) |
| 기술 발전 | AI 매칭, 앱 기반 온디맨드 서비스 확산 |

## 1.3 한국 시장 현황

### 시장 규모
- 맘시터 플랫폼 거래액 기준: 2023년 한 해 **2,600억 원**
- 3대 플랫폼(맘시터, 째깍악어, 자란다) 모두 2021년 이후 대규모 투자 유치
- 민간 돌봄 플랫폼 시장: 프리미엄화, 기술 융합, 서비스 다각화 키워드로 성장 중

### 시장 성장 동력

| 동력 | 세부 내용 |
|------|-----------|
| 맞벌이 가구 증가 | 18세 미만 자녀가 있는 유배우 가구 맞벌이 비중 **58.5%** (2024년 하반기, 역대 최대) |
| 밀레니얼 육아관 | 자녀 양육뿐 아니라 개인 시간도 소중히 여기는 세대 가치관 |
| 정부 지원 확대 | 아이돌봄서비스 예산 **5,134억 원** (2025년), 지원 대상 중위소득 250% 이하로 확대 (2026년) |
| 기술 발전 | AI 매칭 알고리즘, 데이터 기반 맞춤형 돌봄 추천 |
| 플랫폼 경제 확산 | 탄력적 시간제 돌봄 수요 증가 |

### 인구통계

| 연도 | 합계출산율 | 출생아 수 | 비고 |
|------|-----------|-----------|------|
| 2023년 | 0.72명 | 약 23만 명 | 역대 최저 |
| 2024년 | **0.75명** | **238,300명** | 9년 만에 반등 (+3.6%) |
| 2025년 | **0.80명** | - | 4년 만에 0.8명대 회복 |

**맞벌이 가구 현황 (2024년 하반기):**
- 전체 맞벌이 비중: 48.0% (608만 6천 가구)
- 18세 미만 자녀 유배우 가구 맞벌이 비중: **58.5%** (역대 최대)
- 30대 맞벌이: 61.5% / 40대 맞벌이: 59.2%

> **핵심 인사이트**: 아이가 있는 가구 10쌍 중 약 6쌍이 맞벌이이며, 이 비율은 계속 상승 중. 이는 돌봄 서비스 수요의 구조적 성장 동력이다.

**육아휴직 현황:**
- 2024년 육아휴직자: **206,226명** (역대 최대)
- 아빠 육아휴직자: 60,117명 (29.2%)
- 남성 육아휴직 비중: 36.8% (2025년 1~9월)

## 2.1 맘시터 (Momsitter) - 국내 최대 아이돌봄 매칭 플랫폼

### 기업 개요

| 항목 | 내용 |
|------|------|
| **운영사** | 주식회사 맘편한세상 (Happymomworld Inc.) |
| **설립일** | 2016년 12월 6일 |
| **대표이사** | 정지예 |
| **본사** | 서울특별시 서초구 서초대로46길 25, 3층 |
| **임직원 수** | 33명 (2026년 2월) |

### 비즈니스 모델 및 수익 구조

아이돌봄이 필요한 부모와 돌봄을 제공할 시터를 연결하는 **양면 매칭 플랫폼**.

**수익 모델:**
- **이용권 판매**: 부모 회원이 시터에게 면접 제안/지원 수락 시 이용권 구매 필요
- **수수료**: 시터 비용의 **0~5%** (업계 최저)
- **프리미엄 서비스**: 관리형 돌봄 서비스(프로케어, 하이시터, 산후도우미)

**핵심 전략**: 매칭 수수료 최소화 → 회원 규모 극대화 → 관리형 프리미엄 서비스로 수익성 확보

### 주요 서비스 라인업

1. **맘시터 (기본 매칭)** - 시간제 베이비시터 매칭, 전국 평균 시급 약 12,000원
2. **맘시터 프로케어 (관리형)** - 전담 매니저가 시터 매칭~관리까지 원스톱
3. **하이시터 (신생아 전문 회원제)** - 생후 2~48개월 대상 풀타임 돌봄, 40여 가지 상담 항목, 심층 검증
4. **맘시터 산후도우미** - 산후조리원 퇴소 후 생후 100일까지 케어
5. **맘시터 비즈케어 (B2B)** - 기업 임직원 복지 돌봄 서비스

### 사업 실적 및 재무

| 지표 | 수치 |
|------|------|
| **누적 회원 수** | **146만 명** (2025년) |
| **시터 회원** | 약 89만 명 |
| **누적 매칭 수** | 363만 건 |
| **플랫폼 거래 추산액 (2023년)** | 2,600억 원 |
| **2023년 매출** | 전년 대비 4배 성장, **흑자 전환 성공** |
| **2024년 1분기 매출** | 140억 원 (전년 동기 대비 2배 이상) |
| **2024년 1분기 영업이익** | 25억 원 (역대 최대) |
| **2024년 연간 매출** | 약 **318억 원** (전년 대비 44.8% 증가) |

### 투자 이력

| 라운드 | 시기 | 금액 | 주요 투자자 |
|--------|------|------|-------------|
| 초기 | 2019년 | 30억 원 | - |
| 시리즈 B | 2021.09 | 100억 원 | KDB산업은행, KTB네트워크, 디티앤인베스트먼트 등 |
| 시리즈 B+ | 2026년(진행) | 50억 원 추가 유치 중 | - |
| **누적** | - | **130억 원+** | 총 9개 투자사 |

### 안전/신뢰 시스템
- **10단계 시터 인증**: 등·초본, 가족관계증명서, 학적증명서, 교사자격증, 건강확인증 등
- 적성검사 실시, "문제" 시터/부모 **5년간 이용 금지**
- 등록 특허 **2건** (아동 인식 및 행동 분석)

### USP
- 국내 최대 규모 (146만 회원, 89만 시터)
- 업계 최저 수수료 (0~5%)
- 기본 매칭~프리미엄 관리형까지 풀 라인업
- B2B 기업 복지 (100% 재계약률)

## 2.2 째깍악어 (TicTocCroc) - 온오프라인 통합 돌봄 생태계

### 기업 개요

| 항목 | 내용 |
|------|------|
| **운영사** | 주식회사 커넥팅더닷츠 |
| **설립일** | 2016년 9월 7일 |
| **대표이사** | 김희정 (중앙대 경영학과, 연세대 MBA, 한국존슨앤드존슨·리바이스·매일유업 출신) |
| **임직원 수** | 82명 (2026년 3월) |

### 비즈니스 모델

**온라인 매칭 + 오프라인 돌봄 공간** 결합한 **O2O 통합 돌봄 생태계**.

**수익 모델:**
- **온라인 매칭 수수료**: 돌봄비의 **15~20%**
- **오프라인 시설(째깍섬) 이용료**: 멤버십 및 프로그램 참가비
- **B2G 공공조달**: 정부/지자체 돌봄 사업 수주 (13건)
- **사업 다각화**: 펫케어, 시니어 돌봄

### 주요 서비스

1. **째깍악어 앱** - 1~13세 돌봄·교육 교사 매칭, 시간당 약 20,000원
2. **째깍섬 (오프라인 돌봄 공간)** - 전국 **7개소** (잠실, 일산, 판교, 하남, 안양, 수원 등)
   - 멤버십: 1개월 5만원 / 3개월 12만원 / 6개월 20만원 / 1년 30만원
3. **모그와이 (펫 돌봄)** - 도그메이트 인수 후 리브랜딩, 의성 펫월드(1만 2천 평) 위탁운영
4. **시니어 교육/돌봄** - 초고령화 사회 대응

### 안전/검증 시스템
- 신원 증빙, 성범죄 이력 조회, 인적성 검사
- 돌봄 교육 이수 필수, **1:1 모의 돌봄 면접**
- 업계 유일 **전원 동영상 프로필** 촬영, 책임 보증금 납부

### 사업 실적

|------|------|
| **부모 회원** | 37만 명 |
| **교사 회원** | 20만 명 (활동 교사 약 8,000명) |
| **2024년 연간 매출** | **125억 원** |
| **2025년 1분기 매출** | 45억 원 (전년 동기 대비 **173%** 증가) |
| **연평균 매출 성장률** | 99% (9년 연속 성장) |
| **수익성** | 2022~2024년 적자폭 70% 개선, 흑자 전환 임박 |

## 2.3 자란다 (Jaranda) - 데이터 기반 교육·돌봄 매칭

### 기업 개요

| 항목 | 내용 |
|------|------|
| **운영사** | 주식회사 자란다 |
| **설립일** | 2016년 6월 20일 |
| **창업자** | 장서정 (현 비상무이사, 모토로라 12년+제일기획 3년 출신) |
| **현 대표** | 김성환 (2024.05 선임) |
| **임직원 수** | 18명 (2026년 2월, 급감) |

### 비즈니스 모델

**4~13세 아이** 대상 **교육+돌봄 융합 데이터 기반 매칭 플랫폼**.

- **매칭 수수료**: 교사 수업료에서 수수료 부과
- **B2B 기업복지**: **1,100여 개 기업** 도입
- **B2G 공공사업**: 2건

### 주요 서비스
1. **방문 돌봄** - 등하원 도우미, 시간제 돌봄 (시급 약 10,000원~)
2. **방문 수업** - 영어, 수학, 예체능 등 1:1 맞춤 교육
3. **긴급 돌봄** - 1회~정기 방문 유연 선택
4. **자란다 비즈니스 (B2B)** - 기업 임직원 자녀 돌봄 복지

### 기술 특징
- **국내 유일 '성장 데이터'**: 아이 관심사, 학업 성취도 분석
- **텍스트 마이닝 기반 추천 알고리즘**: 매칭 만족도 **97%**
- 1만여 개 활동, 6,000여 개 온라인 놀이클래스

### 사업 실적

| 지표 | 수치 |
|------|------|
| **가입 교사** | 19만 명 |
| **B2B 도입 기업** | 1,100여 개 |
| **2024년 매출** | 약 67.7억 원 (**전년 대비 25% 하락**) |
| **누적 투자** | **448억 원** (업계 최대) |

### 현황 및 과제
- 2024년 매출 하락, 임직원 18명으로 급감 → 사업 구조 조정 신호
- 448억 원 투자에도 수익성 확보가 핵심 과제
- 창업자 비상무이사 전환

| 대학 캠퍼스 침투 | 게시판, 동아리 제휴, 캠퍼스 앰배서더 |
| 교육/자격증 연계 | 시터 교육 프로그램, 수료증 발급 |
| 경제적 인센티브 | 가입 보너스, 추천 리워드, 초기 시급 보장 |
| 경력 경로 제공 | 시터 → 전문 나니 → 교육 전문가 |
| 유연한 근무 | 시간/요일 자유, 즉시 수락/거절 |

### 수요 측면 (부모)

| 전략 | 실행 방법 |
|------|-----------|
| 소셜 네트워크 추천 | 친구 추천 시터, 추천인 할인 |
| 육아 커뮤니티 제휴 | 맘카페, 육아 블로거, 인플루언서 |
| 기업 복지 제휴 | B2B 유입 (높은 LTV) |
| 정부 보조금 연계 | 가격 저항 해소 |
| 무료 체험 | 첫 예약 할인/무료 |

> 63% 부모가 보육시설에서 기술적 요소를 원함, 50%가 온라인으로 시터 탐색

## 5.3 네트워크 효과

```
[직접] 시터↑ → 부모에게 더 많은 선택지 → 부모↑
[간접] 부모↑ → 시터 수입↑ → 시터↑ → 품질↑
[데이터] 매칭↑ → AI 추천 정확도↑ → 만족도↑ → 이용↑
[로컬] 지역 밀도↑ → 빠른 매칭 → 즉시 예약 → 지역 지배
```

**유동성 공식**: 수요 충족률 x 매칭 속도 x 서비스 품질

# Part 7. 경쟁 전략

## 7.1 Porter's Five Forces

| 경쟁력 | 강도 | 분석 |
|--------|------|------|
| **기존 경쟁자** | **높음** | 맘시터/째깍악어/자란다 + 대형 교육기업 진입 |
| **신규 진입자** | **중간~높음** | 기술 진입장벽 낮으나 네트워크 효과로 후발 불리 |
| **대체재** | **높음** | 가족/이웃, 어린이집, 정부 돌봄, 공동육아 |
| **공급자 교섭력 (시터)** | **중간~높음** | 전환 비용 낮음, 인력 부족 심화 |
| **구매자 교섭력 (부모)** | **높음** | 대안 다수, 가격 민감, 전환 비용 낮음 |

> "텍스트북적 실패 시장" (재닛 옐런) - 부모 지불 의향 vs 시터 보수 기대의 구조적 격차

## 7.2 경쟁 해자(Moat) 구축

### 네트워크 효과 해자
```
[사용자 수] → [데이터 축적] → [AI 매칭 정확도] → [만족도]
     ↑                                              ↓
     ← ← ← ← ← [입소문/추천] ← ← ← ← ← ← ← ← ← ←
```

### 4가지 해자

| 해자 | 설명 |
|------|------|
| **네트워크** | 로컬 밀도, 리뷰 축적, 소셜 그래프 |
| **데이터** | 수백만 매칭 기반 AI, 성과/평가 이력, 수요/공급 패턴 |
| **브랜드/신뢰** | 안전 사고 제로, 미디어 PR, 기업 레퍼런스 |
| **규제** | 정부 인증(A등급), 사업 수주, 준수 시스템 비용 |

## 7.3 수직 통합 기회

| 영역 | 사례 |
|------|------|
| 교육/자격증 | 맘시터 시터교육(A등급) |
| 배경조사 | Checkr, Persona 파트너십 |
| 보험 | 돌봄 배상책임보험 패키지 |

## 7.4 B2B vs B2C

### B2C
| 장점 | 단점 |
|------|------|
| 큰 TAM | 높은 CAC, 낮은 LTV |
| 바이럴 가능 | 가격 민감 |
| 빠른 피드백 | 이탈률 높음 |

### B2B
| 장점 | 단점 |
|------|------|
| 높은 계약 금액 | 긴 영업 주기 |
| 낮은 이탈률 (맘시터 100% 재계약) | 커스터마이징 |
| 안정적 반복 수익 | 소수 고객 의존 |

**Bright Horizons B2B**: 1,450+ 기업, Fortune 500 중 220+, 백업케어 $725M+
**맘시터 B2B**: 크래프톤/현대차/토스 등, 100% 재계약, 서울형 아이돌봄비 3년 연속 1위

# Part 10. 실패 사례 및 교훈

## 10.1 Poppy (2015~2018, 시애틀)

- Y Combinator 출신, $2M+ 투자, 36,000건 예약 달성 후 폐업
- **실패 원인**:
  1. 부모 지불 의향 vs 시터 보수 기대 격차를 기술로 메울 수 없음
  2. 초기 열성 사용자의 만족도가 대중 시장으로 이전되지 않음
  3. 높은 개인화 → 높은 운영비 → 높은 가격 → 이탈 악순환
- **교훈**: "좋은 서비스"와 "지속 가능한 비즈니스"는 다른 문제

## 10.2 Nanno (덴버, ~2022)

- 이벤트/행사 중심 베이비시팅
- **실패 원인**: 크라우드펀딩 직후 COVID-19 → 대면 이벤트 중단 → 핵심 수요 소멸
- **교훈**: 이벤트 기반 모델은 외부 충격에 취약, 타이밍이 성패 좌우

## 10.3 Care.com 신뢰 위기 (2019)

- WSJ 탐사보도: 범죄기록 시터 추천, 비인가 시설 허위 인가
- 6년간 절도, 아동학대, 성폭행, 살인 사건 발생
- $500K + $700K + FTC 제재
- **교훈**: 배경조사는 유료 옵션이 아닌 필수 — 안전 비용 절감은 기업 존속 위협

## 10.4 공통 실패 요인

| 실패 요인 | 빈도 |
|-----------|------|
| 유닛 이코노믹스 실패 (지불 의향 vs 보수 격차) | 매우 높음 |
| 신뢰/안전 문제 | 높음 |
| 닭과 달걀 미해결 | 높음 |

# Part 13. 전략적 액션 아이템

## 즉시 적용 (0~6개월)

1. **프리미엄 계층화**: 무료 → 기본 구독 → 프리미엄 관리형 3단계 수익 구조
2. **공급 우선 확보**: 대학 캠퍼스 침투 + 시터 교육 프로그램
3. **지역 집중**: 한 동네/구에서 유동성 확보 후 점진적 확장
4. **신뢰 우선**: 배경조사를 비용이 아닌 투자로 (Care.com 교훈)
5. **양면 리뷰**: 동시 공개 방식의 양면 리뷰 시스템

## 중기 전략 (6~18개월)

6. **B2B 진출**: 기업 복지 돌봄 서비스로 안정적 수익원
7. **정부 사업**: 아이돌봄비 지원사업 등 정부 채널 활용
8. **AI 매칭 고도화**: 데이터 축적에 따른 알고리즘 개선
9. **프로케어 모델**: 관리형 서비스로 프리미엄 세그먼트

## 장기 전략 (18개월+)

10. **생태계 확장**: 시니어케어, 펫케어, 가사도우미
11. **오프라인 통합**: 체험/돌봄 공간 (째깍섬 모델)
12. **데이터 수익화**: 익명화 돌봄 트렌드 인사이트
13. **국제 확장**: 아시아 태평양 (가장 빠른 성장 지역)

## 핵심 성공 요인

1. **시터 품질 관리**: 검증된 시터 확보·교육이 플랫폼 경쟁력의 핵심
2. **수익 모델 다각화**: 단순 매칭 수수료를 넘어 관리형·B2B·오프라인
3. **기술 투자**: AI 매칭, 안전 기술, 데이터 분석
4. **규제 대응**: 2026년 국가자격제/등록제 선제 대응
5. **브랜드 신뢰**: 안전 사고 예방 및 위기 관리
