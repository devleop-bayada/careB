# 1. 서비스 개요

## 1.1 서비스 비전 및 미션

### 비전
**"모든 어르신이 존엄하고 따뜻한 돌봄을 받을 수 있는 세상"**

초고령사회 진입(2025년 고령인구 20.6%)과 함께 폭발적으로 증가하는 노인 돌봄 수요에 대응하여, 검증된 요양보호사와 돌봄이 필요한 어르신·보호자를 신뢰 기반으로 연결하는 대한민국 대표 시니어 돌봄 매칭 플랫폼이 된다.

### 미션
**"신뢰할 수 있는 요양보호사를 가장 빠르고 쉽게 찾을 수 있도록 한다"**

- 보호자에게: 검증된 요양보호사를 투명하게 비교하고 안심하고 맡길 수 있는 환경 제공
- 요양보호사에게: 더 많은 일자리 기회, 공정한 보수, 커리어 성장 경로 제공
- 기관에게: 인력 수급 효율화 및 서비스 품질 향상 지원

## 1.2 서비스 핵심 가치 (Value Proposition)

### 보호자 관점
| 가치 | 설명 |
|------|------|
| **신뢰** | 자격증 검증 + 경력 인증 + 실사용 리뷰로 믿을 수 있는 요양보호사 |
| **투명성** | 시간당 요금, 경력, 전문 분야를 한눈에 비교 |
| **편의성** | 앱 하나로 검색 → 상담 → 예약 → 결제 → 돌봄일지 확인까지 원스톱 |
| **안전망** | 돌봄 중 사고 보험, 24시간 긴급 지원, 부정행위 보호 시스템 |
| **맞춤 매칭** | AI 기반 어르신 상태·요구사항에 최적화된 매칭 |

### 요양보호사 관점
| 가치 | 설명 |
|------|------|
| **소득 향상** | 직접 계약으로 중간 마진 최소화, 더 많은 수입 |
| **유연 근무** | 원하는 시간·지역·돌봄 유형 선택 |
| **커리어 성장** | 교육 콘텐츠, 전문 자격 취득 지원, 등급 시스템 |
| **보호** | 부당 처우 신고 시스템, 표준 계약서, 플랫폼 보호 |
| **커뮤니티** | 동료 요양보호사 네트워크, 정보 공유 |

### 기관 관점
| 가치 | 설명 |
|------|------|
| **인력 수급** | 검증된 요양보호사 인력풀 즉시 접근 |
| **관리 효율** | 출퇴근 기록, 돌봄일지, 정산 자동화 |
| **품질 관리** | 실시간 서비스 모니터링 및 피드백 |

## 1.3 서비스 슬로건

**"어르신의 내일을, 믿음으로 잇다"**

서브 슬로건:
- 보호자 대상: "검증된 요양보호사, 이제 바야다에서 찾으세요"
- 요양보호사 대상: "당신의 돌봄이 더 빛나는 곳, 바야다"
- 기관 대상: "인력 수급부터 품질 관리까지, 바야다 비즈니스"

# 2. 타겟 유저 분석

## 2.1 보호자 페르소나

### 페르소나 1: 박민준 (직장인 아들, 43세)

**기본 정보**
- 직업: IT 대기업 부장, 주 60시간 근무
- 가족: 배우자(직장인), 자녀 2명(초등), 어머니(79세, 치매 초기)
- 거주: 성남시 분당구 (어머니는 서울 도봉구 독거)
- 월 소득: 850만 원

**상황**
어머니가 2년 전 치매 진단을 받았고, 현재 장기요양 3등급이다. 혼자 사는 어머니를 매일 돌볼 수 없어 요양보호사를 찾고 있지만, 인터넷 검색으로는 어디를 믿어야 할지 모르겠다. 지역 요양원에 문의했더니 등록이 오래 걸린다고 했고, 주변 지인 소개로 한 번 써봤는데 요양보호사가 갑자기 그만두어 고생했다.

**니즈**
- 치매 어르신 전문 요양보호사 빠르게 찾기
- 어머니 상태를 실시간으로 확인할 수 있는 방법
- 요양보호사 갑작스러운 결근 시 대체 인력 즉시 확보
- 장기요양보험 청구 행정 처리 대행
- 믿을 수 있는 검증 체계

**Pain Point**
- 요양보호사 자격은 있는데 치매 돌봄 경험이 실제로 있는지 확인 불가
- 플랫폼 없는 직거래는 불안하고, 요양원은 시설 입소가 싫음
- 어머니가 새로운 사람에게 거부 반응을 보여 신중하게 선택해야 함
- 일정 관리와 결제, 기록이 각각 따로라 관리가 번거로움

**Jobs-to-be-Done**
- 기능적: "검증된 치매 전문 요양보호사를 2일 안에 매칭받고 싶다"
- 감성적: "어머니가 낯선 사람에게 잘 적응할 수 있을지 사전에 알고 싶다"
- 사회적: "요양보호사에게 어머니를 안심하고 맡기는 책임감 있는 아들이 되고 싶다"

**기본 정보**
- 직업: 전업주부 (전직 간호사, 5년 경력)
- 가족: 남편(직장인), 자녀 1명(유치원), 시아버지(82세, 뇌졸중 후유증)
- 거주: 동거 (시아버지와 같은 집)
- 월 가구 소득: 600만 원

**상황**
시아버지가 1년 전 뇌졸중으로 쓰러진 후 반신마비가 왔다. 며느리인 수진 씨가 주 돌봄자인데, 간호사 경력이 있어 의료적 처치는 가능하지만 체력적으로 한계를 느끼고 있다. 시아버지는 장기요양 1등급을 받았고, 주 5일 방문요양 서비스를 받고 있지만 기관에서 보내주는 요양보호사의 실력 편차가 크고, 불만이 있어도 기관에 말하기가 불편하다.

**니즈**
- 현재 이용 중인 기관 요양보호사 외 추가 시간제 서비스
- 본인이 선택하고 평가할 수 있는 구조
- 방문요양 + 방문목욕 + 물리치료 연계까지 한 곳에서 관리
- 돌봄 일지를 남편도 볼 수 있게 공유

**Pain Point**
- 기관에서 일방적으로 요양보호사를 배정하고 바꿀 수 없음
- 요양보호사의 서비스 내용을 확인할 방법이 없음
- 직접 돌봄에 지쳐도 "내가 며느리인데..." 하는 죄책감이 있음
- 남편이 돌봄 상황을 잘 모르고 협력을 이끌어내기 어려움

**Jobs-to-be-Done**
- 기능적: "내가 직접 선택한 요양보호사와 1:1로 계약하고 싶다"
- 감성적: "시아버지 돌봄 부담을 남편과 함께 나누고 싶다"
- 사회적: "전문적으로 돌봄을 관리하는 스마트한 보호자로 인정받고 싶다"

### 페르소나 3: 김혜경 (원거리 딸, 56세)

**기본 정보**
- 직업: 자영업(카페 운영)
- 가족: 미혼, 부모님 모두 생존(아버지 84세 신장투석, 어머니 81세 건강)
- 거주: 부산 (부모님은 서울 강동구)
- 월 소득: 약 400만 원

**상황**
멀리 살다 보니 부모님 상황을 실시간으로 파악하기 어렵다. 아버지가 주 3회 혈액투석을 받아야 하는데, 병원 이동 동행이 가장 큰 문제다. 가끔 올라가서 상황을 보면 어머니도 지쳐 있고, 아버지도 식사·약 복용 관리가 잘 안 되고 있다. 친척들에게 부탁하기도 민망하고, 지역 요양원은 비용이 너무 비싸다.

**니즈**
- 병원 동행 서비스 (주 3회)
- 식사 준비 및 약 복용 확인 서비스
- 부모님 상태를 원격에서 사진/동영상으로 확인
- 긴급 상황 시 즉시 연락 받는 알림 시스템
- 비용 예측 가능성 (월정액 or 패키지 요금)

**Pain Point**
- 원거리로 인해 어떤 사람이 부모님 집에 오는지 직접 확인 불가
- 여러 서비스(병원 동행, 식사, 투약 관리)를 각각 다른 곳에서 찾아야 함
- 서비스 이용 중 불상사가 생겼을 때 즉시 대응 불가
- 장기요양보험 외 사설 비용 부담이 예측 불가

**Jobs-to-be-Done**
- 기능적: "원격에서도 부모님 돌봄 상황을 매일 확인하고 싶다"
- 감성적: "멀리 있어도 좋은 자식을 하고 있다는 안도감을 갖고 싶다"
- 사회적: "부모님을 전문가에게 맡기는 합리적인 선택을 했다고 인정받고 싶다"

## 2.2 요양보호사 페르소나

### 페르소나 1: 최명숙 (경력자 독립 활동, 52세)

**기본 정보**
- 자격: 요양보호사 1급 (취득 후 8년 경력)
- 현황: 재가요양센터 소속이었으나 퇴직, 개인 활동 시작
- 거주: 서울 노원구
- 전문: 치매, 뇌졸중 어르신 돌봄

**상황**
8년간 재가요양센터에 소속되어 일했는데, 중간 마진이 너무 많이 빠져 실질 급여가 적다는 생각이 들었다. 퇴직하고 혼자 일하려니 어르신 가족들과 직접 연결될 방법이 없다. 주변에 알음알음으로 2가구를 담당하고 있는데 더 확장하고 싶다.

**니즈**
- 내 전문성(치매 전문)을 잘 보여주는 프로필 공간
- 직접 클라이언트와 연결되는 채널
- 일정 관리 도구 (여러 가구 담당 시)
- 수입 정산 및 세금 처리 도움
- 갑작스러운 일정 변경 시 대체 인력 연계

**Pain Point**
- 개인이라 보험 가입이 어렵고 사고 시 보호 장치 없음
- 새 클라이언트를 찾는 채널이 없어 지인 소개에 의존
- 표준 계약서가 없어 분쟁 발생 시 불리한 입장
- 돌봄 일지를 따로 기록해야 하는 번거로움

**Jobs-to-be-Done**
- 기능적: "내 경력과 전문성을 보여주는 프로필로 더 많은 클라이언트를 찾고 싶다"
- 감성적: "전문 요양보호사로서 인정받고 싶다"
- 사회적: "기관에 종속되지 않고 독립적으로 일하는 전문가로 인정받고 싶다"

**니즈**

- 초보자 교육 콘텐츠 및 실무 가이드

# 4. 유저 플로우

## 4.1 보호자 플로우

### 단계 1: 가입 및 어르신 등록

```
앱 다운로드
    ↓
소셜 로그인 선택 (카카오/네이버/구글/애플)
    ↓
휴대폰 번호 인증 (6자리 OTP)
    ↓
기본 정보 입력 (이름, 생년월일)
    ↓
어르신 등록 선택 (본인 / 가족)
    ↓
어르신 정보 입력
  ├─ 성명, 생년월일, 성별
  ├─ 주소 (돌봄 장소)
  ├─ 장기요양 등급 여부
  │   ├─ 있음 → 등급 선택 (1~5등급, 인지지원등급)
  │   └─ 없음 → 등급 신청 안내 또는 사설 서비스 진행
  ├─ 주요 질환 선택 (복수 선택)
  │   ├─ 치매/인지장애, 뇌졸중, 파킨슨, 당뇨, 신장질환
  │   ├─ 암, 심장질환, 골절/낙상, 욕창, 정신질환 등
  └─ 현재 이동 능력 (혼자 가능/부축 필요/완전 의존)
    ↓
서비스 니즈 입력
  ├─ 원하는 서비스 유형 (방문요양/병원 동행/간병 등)
  ├─ 희망 시작일 및 빈도 (주 몇 회, 몇 시간)
  └─ 예산 범위
    ↓
홈 화면으로 이동 (검색 시작 안내)
```

### 단계 2: 요양보호사 검색 및 선택

```
홈 화면 '요양보호사 찾기' 버튼
    ↓
자동 완성된 조건 확인 (어르신 정보 기반)
    ↓
검색 결과 리스트 (기본: 추천 순)
    ↓
[선택A] 리스트에서 프로필 클릭
    ↓
요양보호사 상세 프로필 조회
  ├─ 기본 정보 (경력, 자격증, 전문 분야)
  ├─ 요금 및 가능 일정
  ├─ 이전 이용자 리뷰 및 평점
  └─ 동영상 자기소개 (있는 경우)
    ↓
관심 목록 저장 OR 돌봄 제안 발송
```

### 단계 3: 상담 및 면접

```
돌봄 제안서 작성
  ├─ 어르신 상세 상황 입력 (미입력 항목 보완)
  ├─ 희망 서비스 내용 구체화
  └─ 제안 메시지 작성
    ↓
요양보호사에게 제안 발송
    ↓
요양보호사 확인 (24시간 내 응답)
    ↓
[수락] → 채팅 창 활성화
    ↓
채팅으로 상세 상담
  ├─ 어르신 특이사항 공유
  ├─ 요양보호사 경험 확인
  └─ 일정/요금 세부 조율
    ↓
[선택] 화상 면접 요청 (인앱 화상통화)
    ↓
계약 진행 결정
```

### 단계 4: 계약 및 결제 설정

```
계약 조건 확인
  ├─ 서비스 유형 및 내용
  ├─ 일정 (시작일, 주기, 시간)
  ├─ 요금 및 결제 방식
  └─ 계약 기간
    ↓
표준 계약서 자동 생성
    ↓
계약서 검토 및 전자 서명
    ↓
결제 수단 등록 (카드/간편결제)
    ↓
첫 회차 결제 처리 (에스크로)
    ↓
계약 확정 완료 알림
    ↓
요양보호사 연락처 공개 (계약 완료 후)
```

### 단계 5: 돌봄 진행 및 모니터링

```
돌봄 당일
    ↓
요양보호사 출발 알림 수신
    ↓
방문 체크인 알림 수신 (GPS 자동)
    ↓
돌봄 진행 중
  └─ 요양보호사 서비스 수행 + 일지 실시간 작성
    ↓
중간 사진/메시지 수신 (선택적)
    ↓
방문 체크아웃 알림 수신
    ↓
돌봄일지 확인
  ├─ 서비스 수행 내역
  ├─ 어르신 상태 기록
  ├─ 특이사항
  └─ 복용 약물/식사 기록
    ↓
72시간 이의제기 없으면 자동 결제 확정
```

### 단계 6: 리뷰 및 재이용

```
서비스 완료 후 24시간 내 리뷰 요청 알림
    ↓
평가 작성
  ├─ 별점 (전체/전문성/친절도/시간 준수/의사소통/일지 충실도)
  └─ 텍스트 후기
    ↓
재이용 의향 선택
  ├─ 재이용 → 정기 계약 전환 안내
  └─ 다른 분 찾기 → 검색 화면으로
    ↓
리뷰 공개 (요양보호사 리뷰 작성 완료 후 동시 공개)
```

## 4.2 요양보호사 플로우

### 단계 1: 가입 및 자격 인증

```
앱 다운로드
    ↓
'요양보호사로 가입' 선택
    ↓
휴대폰 본인인증
    ↓
기본 정보 입력 (이름, 생년월일, 주소)
    ↓
자격증 인증 (필수)
  ├─ 요양보호사 자격증 촬영/업로드
  ├─ OCR 자동 인식
  └─ 공단 자격 검증 API 확인 (1~2시간 내)
    ↓
신원 인증
  ├─ 주민등록증 또는 운전면허증 촬영
  ├─ 얼굴 인식 대조
  └─ 통과/미통과 판정
    ↓
범죄경력 조회 동의 및 진행
  └─ 결과 수신 (1~3일 소요, 문제없으면 자동 통과)
    ↓
인증 완료 → '인증 완료' 뱃지 부여
    ↓
프로필 작성 안내로 이동
```

### 단계 2: 프로필 작성

```
프로필 완성도 표시 (0~100%)
    ↓
필수 항목 작성 (완성도 0→60%)
  ├─ 프로필 사진 업로드
  ├─ 자기소개 (100자 이상)
  ├─ 전문 분야 선택 (최대 5개)
  ├─ 보유 자격증 전체 등록
  └─ 희망 근무 조건 설정
    ↓
권장 항목 작성 (완성도 60→100%)
  ├─ 동영상 자기소개 촬영 (30~60초)
  ├─ 경력 증명서 업로드
  ├─ 가용 일정 상세 등록
  └─ 요금 설정 (서비스 유형별)
    ↓
프로필 미리보기 확인
    ↓
활동 상태 ON → 매칭 대기 시작
```

### 단계 3: 돌봄 요청 수락

```
돌봄 요청 푸시 알림 수신
(설정 조건 일치: 지역/시간/서비스 유형)
    ↓
요청 상세 확인
  ├─ 어르신 기본 정보
  ├─ 원하는 서비스 내용
  ├─ 일정 및 보수
  └─ 보호자 기본 정보
    ↓
24시간 내 응답 필요
    ↓
```

```
    ↓

    ↓
```

# 7. 차별화 전략

## 7.1 기존 경쟁사 대비 차별점 분석

### 현재 요양보호사 매칭 플랫폼 문제점

| 플랫폼 | 주요 문제점 |
|--------|-----------|
| **케어닥** | 기관 중심 서비스로 개인 매칭 비중 낮음, UI/UX 복잡 |
| **케어네이션** | 간병인 중심, 요양보호사 전문 기능 부족 |
| **위케어** | 지역 커버리지 제한, 자격 검증 체계 미흡 |
| **요양나라** | 정보 나열 수준, 실제 매칭 기능 부족 |
| **돌봄플러스** | 요양시설 연계 중심, 재가 서비스 약함 |

### 바야다 차별화 포인트 5가지

#### 1) 신뢰 최우선 인증 체계 (Trust-First)
경쟁사 대부분이 자격증 보유 여부만 확인하는 것과 달리, 바야다는 **7단계 인증 시스템**을 도입한다.

```
[바야다 7단계 인증]
1단계: 휴대폰 본인인증
2단계: 주민등록증 신원 확인 + 얼굴 인식
3단계: 요양보호사 자격증 OCR + 공단 API 검증
4단계: 범죄경력 조회 (성범죄, 아동학대, 노인학대)
5단계: 건강진단서 확인 (최근 1년 이내)
6단계: 배상책임보험 가입 확인
7단계: 바야다 자체 인성 평가 (온라인 체크리스트)
```

통과 시 '바야다 인증 완료' 뱃지 부여, 미통과 항목 명시.

#### 2) 어르신 중심 매칭 알고리즘 (Care-Centered Matching)
단순 위치/시간 매칭이 아닌, **어르신의 질환·성격·습관**까지 반영한 정밀 매칭.

```
[매칭 변수]
어르신 측: 질환 유형, 증증도, 이동 능력, 성격 (활동적/조용함),
           과거 돌봄 경험 (이전 요양보호사와의 관계 분석),
           종교·식단 제한, 선호 성별·연령대
요양보호사 측: 전문 분야, 돌봄 경험 데이터, 유사 케이스 성공률,
               위치, 가용 시간, 고객 평점, 재이용률
```

#### 3) 실시간 돌봄 투명성 (Real-Time Transparency)
원거리 보호자나 바쁜 직장인 보호자를 위한 **실시간 돌봄 모니터링**.

- 출퇴근 GPS 자동 기록 및 알림
- 돌봄일지 당일 실시간 작성 (서비스 중 입력)
- 가족 구성원 공유 기능 (최대 5명)
- 이상 징후 자동 알림 (특이사항 기록 시)

#### 4) 보호자-요양보호사 모두에게 공정한 구조 (Fair Platform)
기존 플랫폼이 보호자 또는 기관 중심인 반면, 바야다는 **양면 공정성** 원칙 적용.

- 요양보호사: 표준 계약서, 부당 처우 신고 시스템, 플랫폼 보호
- 보호자: 에스크로 결제, 이의제기 기간, 부정행위 보상
- 수수료: 투명 공개 (요양보호사 3%, 보호자 0%)

#### 5) 돌봄 연속성 보장 (Care Continuity)
한국 요양 시장의 최대 문제인 **요양보호사 갑작스러운 결근/이탈** 해결.

- 비상 대체 인력 시스템: 결근 신고 시 3시간 내 대체 인력 배정
- 정기 계약 우선 매칭: 같은 요양보호사와의 장기 관계 형성 지원
- 인수인계 일지: 요양보호사 교체 시 어르신 정보 완벽 인수인계
- 기관 연계 옵션: 개인 매칭 어려운 경우 제휴 기관으로 연결

## 9.2 AI 매칭 알고리즘 설계

### 1단계 (MVP): 규칙 기반 매칭
```
입력값 → 필터링 → 정렬
- 필수 조건 필터: 지역(반경 5km), 가용 시간, 서비스 유형
- 전문 분야 매칭: 어르신 질환 ↔ 요양보호사 전문 분야
- 정렬 기준: (평점 × 40%) + (경력 × 30%) + (재이용률 × 30%)
```

### 2단계 (6개월 후): 학습 기반 매칭
```
데이터 축적 (매칭 성공/실패, 재이용률, 리뷰 텍스트)
→ 협업 필터링 (비슷한 어르신-요양보호사 패턴)
→ 콘텐츠 기반 필터링 (요양보호사 특성 벡터)
→ 하이브리드 추천
```

### 매칭 성공률 목표
- MVP 출시: 매칭 성공률 65%
- 6개월: 78%
- 1년: 85%+
