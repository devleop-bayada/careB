## Executive Summary

**바야다(BAYADA)**는 한국의 구조적 초고령화를 배경으로, 요양보호사와 돌봄 수요자(가족/수급자)를 직접 연결하는 **디지털 재가요양 매칭 플랫폼**이다.

### 핵심 기회 요인

| 지표 | 현황 | 5년 후 전망 |
|------|------|-------------|
| 65세 이상 인구 | 1,024만 명 (20.6%) | 약 1,500만 명 (30%+) |
| 장기요양 수급자 | 약 110만 명 | 200만 명+ |
| 장기요양보험 총급여비 | 12조 원 | 20조 원+ |
| 재가요양 시장 | 약 7조 원 | 12조 원+ |
| 활동 요양보호사 | 55만 명 | 80만 명+ (예상) |

### 비즈니스 모델 요약

- **수익원 1**: 매칭 수수료 (요양보호사 시급의 8~12%)
- **수익원 2**: 보호자 프리미엄 구독 (월 9,900원~29,900원)
- **수익원 3**: B2B 기업 복지 서비스 (요양·간병 복지 패키지)
- **수익원 4**: B2G 지자체·국민건강보험공단 연계 사업

### 3년 목표

- 누적 요양보호사 회원: 15만 명
- 누적 보호자/수급자 회원: 5만 명
- 연간 거래액(GMV): 1,200억 원
- 연간 매출: 120억 원 (GMV의 10%)
- 영업이익: 흑자 전환 (36개월 시점)

## 1. 시장 분석

### 1.1 한국 장기요양보험 제도 분석

#### 제도 개요

한국 장기요양보험 제도는 2008년 7월 시행된 **사회보험 방식**의 노인 돌봄 제도로, 건강보험과 별도로 운영된다.

| 항목 | 내용 |
|------|------|
| 운영 주체 | 국민건강보험공단 |
| 재원 구성 | 장기요양보험료(가입자 부담) + 국가 지원(20%) + 본인 부담(15~20%) |
| 수급 대상 | 65세 이상 또는 노인성 질병을 가진 65세 미만 |
| 등급 체계 | 1등급(최중증)~5등급 + 인지지원등급(총 6단계) |
| 2024년 수급자 | 약 110만 명 (65세 이상 인구의 약 11%) |
| 2024년 총급여비 | 약 12조 원 |

#### 급여 유형 및 월 한도액 (2025년 기준)

| 등급 | 재가급여 월 한도액 | 시설급여 월 한도액 |
|------|-------------------|-------------------|
| 1등급 | 2,069,900원 | 시설 비용 (요양원 입소) |
| 2등급 | 1,869,600원 | 시설 비용 |
| 3등급 | 1,455,800원 | - |
| 4등급 | 1,341,800원 | - |
| 5등급 | 1,151,600원 | - |
| 인지지원등급 | 638,800원 | - |

**재가급여 세부 유형:**
- **방문요양**: 요양보호사가 수급자 가정 방문, 신체활동 + 가사지원
- **방문목욕**: 목욕 장비 탑재 차량으로 방문
- **방문간호**: 간호사/간호조무사 방문 의료 서비스
- **주야간보호**: 낮 시간 시설 이용 후 귀가
- **단기보호**: 일정 기간 시설 입소

#### 제도의 구조적 문제점 (플랫폼 기회 창출 지점)

1. **공급자 정보 비대칭**: 수급자 가족이 어느 요양기관이 좋은지 알기 어려움
2. **매칭 비효율**: 지역 요양기관 전화 문의 → 대기 → 재배정의 전근대적 프로세스
3. **요양보호사 처우 문제**: 월 150~180만 원 수준, 업무 강도 대비 낮은 처우 → 이직률 연 30%+
4. **불투명한 서비스 품질**: 표준화된 품질 지표 없음, 사고 발생 시 책임 소재 불명확
5. **재가요양 저변 확대**: 정부 커뮤니티케어 정책으로 시설보다 재가 선호 유도

### 1.2 TAM / SAM / SOM 산출

#### TAM (Total Addressable Market) — 전체 주소 가능 시장

```
장기요양 재가서비스 총 급여비 (2024): 약 7조 원
+ 장기요양 미수급 민간 요양·간병 시장 (추산): 약 3~5조 원
+ 가사·생활지원 서비스 시장: 약 1조 원
= TAM ≈ 11~13조 원 (2024년 기준)
2030년 TAM 전망: 약 20~25조 원 (연 10~12% 성장)
```

| 구분 | 2024년 | 2027년 | 2030년 |
|------|--------|--------|--------|
| 장기요양 재가급여 | 7조 원 | 9.5조 원 | 13조 원 |
| 민간 요양·간병 | 4조 원 | 5.5조 원 | 7조 원 |
| 가사·생활지원 | 1조 원 | 1.5조 원 | 2조 원 |
| **TAM 합계** | **12조 원** | **16.5조 원** | **22조 원** |

#### SAM (Serviceable Addressable Market) — 서비스 가능 시장

바야다가 현실적으로 접근 가능한 시장:

```
재가방문요양 + 민간 간병 + 온라인 매칭 전환 가능 수요
= 방문요양 급여비 약 4.5조 원의 디지털화 전환 가능분 (40%): 1.8조 원
+ 민간 간병 시장 디지털 매칭 가능분 (30%): 1.2조 원
= SAM ≈ 3조 원 (2024년 기준)
```

#### SOM (Serviceable Obtainable Market) — 실제 획득 가능 시장

```
초기 3년 내 목표 시장 점유율:
- 수도권 재가요양 매칭 시장의 3~5% 점유
- GMV 기준: 1,200억 원 (3년 목표)
= SOM ≈ 1,000~1,500억 원 (3년 시점 연간 기준)
```

### 1.4 시장 성장률 및 전망

#### 성장 동인 5가지

1. **인구 구조 변화 (불가역적)**: 2025년 65세 이상 20.6% → 2035년 30%+ → 2050년 40%+ 예상. 고령화는 되돌릴 수 없는 구조적 수요다.

2. **장기요양 등급 확대**: 정부는 지속적으로 수급 대상을 확대 중. 2008년 수급자 22만 명 → 2024년 110만 명 (5배 성장). 2030년 200만 명 돌파 전망.

3. **재가요양 정책 강화**: 정부 커뮤니티케어(지역사회통합돌봄) 로드맵에 따라 시설 입소보다 재가 서비스 확대 유도. 재가요양 급여 인상 기조 유지.

4. **요양보호사 부족 심화**: 수요 증가 대비 활동 인력 부족. 2024년 기준 자격 취득자 250만 명 중 활동자 55만 명. 미활동 자격자 195만 명이 잠재 공급 풀.

5. **디지털 전환 가속**: 중장년 보호자(40~60대)의 스마트폰 사용률 93%+. 코로나 이후 비대면 서비스 수용성 대폭 상승.

#### 시장 규모 성장 예측

| 연도 | 수급자 수 | 장기요양 급여비 | 재가요양 비중 | 재가요양 시장 |
|------|-----------|----------------|--------------|--------------|
| 2024 | 110만 명 | 12조 원 | 60% | 7.2조 원 |

### 2.3 Porter's Five Forces 분석

#### 1. 신규 진입자 위협 (Threat of New Entrants) — 중간 (★★★☆☆)

**진입 장벽:**
- 요양보호사·수급자 양면 네트워크 구축 비용 (닭과 달걀 문제)
- 장기요양보험 급여 청구 시스템 이해 필요
- 신뢰 구축을 위한 브랜드 투자 기간 (최소 12~18개월)
- 안전사고 발생 시 법적 책임 구조 복잡성

**진입 완화 요인:**
- 앱 개발 비용 하락 (SaaS 플랫폼 활용 가능)
- 케어닥 성공으로 시장 존재 증명됨 → 투자 유치 용이
- 정부 스타트업 지원 프로그램 다수

---

#### 2. 기존 경쟁자 간 경쟁 (Competitive Rivalry) — 높음 (★★★★☆)

- 케어닥, 케어네이션, 위케어 등 다수 플레이어
- 아직 지배적 1위 플레이어 없음 → **Winner-takes-most 경쟁 중**
- 맘시터(아동 돌봄) 등 인접 플랫폼의 시니어 진출 가능성

---

#### 3. 구매자 교섭력 (Bargaining Power of Buyers) — 중간 (★★★☆☆)

- 수급자/보호자: 선택지 증가로 교섭력 강화 중
- but 신뢰·안전 중심 의사결정 → 브랜드 충성도 형성 가능
- 장기요양보험 수급자: 급여 한도 내 소비 → 가격 민감도 낮음

---

#### 4. 공급자 교섭력 (Bargaining Power of Suppliers) — 높음 (★★★★☆)

- 요양보호사: 활동 인력 부족 심화 → 교섭력 점점 강화
- 자격증 보유자 중 활동률 22%에 불과 → 활성화 시 공급 잠재력 높음
- 요양보호사 이직률 연 30%+ → 공급 불안정

**대응 전략**: 요양보호사 소득·처우 개선을 통한 플랫폼 락인

---

#### 5. 대체재 위협 (Threat of Substitutes) — 중간 (★★★☆☆)

- 요양원 입소: 월 150~300만 원, 대체재이나 비선호
- 가족 직접 돌봄: 가능하나 맞벌이 증가로 어려움
- 정부 아이돌봄 서비스 유사 공공 요양 서비스: 대기 시간 길고 배정 어려움

### 3.1 초기 타겟 시장 및 지역

#### 1차 타겟: 서울시 + 경기도 (수도권)

**이유:**
- 전국 장기요양 수급자의 약 45% 집중
- 재가요양 서비스 수요 밀도 가장 높음
- 디지털 서비스 수용성 최고
- 요양보호사 공급 집중
- 투자자·파트너·미디어 접근성 용이

**수도권 세부 우선 순위:**
1. 서울 강남구/서초구/송파구 (고소득, 프리미엄 서비스 수용)
2. 서울 노원구/강북구/도봉구 (수급자 밀집, 대규모 수요)
3. 경기도 성남/용인/수원 (신도시, 중산층 수요)

#### 2차 타겟 (6~18개월): 5대 광역시

부산, 대구, 인천, 광주, 대전 — 수도권 모델 복제 적용

#### 타겟 고객 페르소나

**공급자 페르소나 — "활동하고 싶은 민경숙씨 (52세)"**
- 요양보호사 자격증 취득 5년, 현재 미활동
- 전 직장: 오프라인 요양기관, 월 160만 원 받고 그만뒤
- 불만: 열악한 처우, 장거리 이동, 불투명한 수당 정산
- 바라는 것: 집 근처 수급자, 적정 시급, 정기적 수입, 안전한 환경
- 디지털 기기: 스마트폰 일상 사용, 카카오톡 주 사용 플랫폼

**수요자 페르소나 — "부모님 걱정하는 이준호씨 (48세)"**
- 직장인, 서울 거주, 부모님은 경기도 수원 거주
- 어머니(76세) 장기요양 3등급, 방문요양 필요
- 불만: 요양기관 찾기 어렵고, 요양보호사 품질 불투명
- 바라는 것: 신뢰할 수 있는 요양보호사, 돌봄 현황 실시간 확인
- 디지털 기기: 스마트폰 헤비 유저, 카카오페이 사용

| 미활동 자격자 재진입 | 방법 몰라서 미활동 | 간편 등록, 원하는 시간만 |

#### 공급 채널 전략

**1. 요양보호사 교육기관 제휴 (가장 효과적)**
- 전국 요양보호사 교육기관: 약 500개소
- 수료생에게 바야다 직접 등록 유도
- 제휴 조건: 첫 3개월 수수료 0% + 교육기관 소개 수수료 건당 5만 원

### 3.5 닭과 달걀 문제 해결 방안

**문제**: 요양보호사가 없으면 보호자가 안 오고, 보호자가 없으면 요양보호사가 안 온다.

#### 해결 전략: 공급 우선 집중 + 수요 시뮬레이션

**Step 1: 공급을 먼저 모아라 (0~3개월)**
- 요양보호사 1,000명 사전 등록 (수수료 없는 기간 보장)
- 요양보호사에게 "대기 중에도 교육 콘텐츠 제공" → 머물 이유 부여
- 단, 수요 없어도 공급 유지 → 무료 프로필 노출 + 정보 제공

**Step 2: 지역 집중으로 밀도 확보 (1~2개월)**
- 서울 강남구 반경 5km 집중 → 밀도 높은 매칭 가능
- "이 지역은 요양보호사가 충분하다"는 인식 → 보호자 유입

**Step 3: 앵커 파트너 활용**
- 강남 지역 병원 1~2곳과 "퇴원 환자 재가요양 연계" 협약
- 병원 사회복지사가 바야다로 직접 보호자 연결
- 초기 수요의 30~40%를 앵커 파트너에서 확보

**Step 4: 조기 후기 구축**
- 초기 50건 매칭: CS팀 수동 관리, 만족도 극대화
- 보호자 후기 적극 수집 → 플랫폼 신뢰도 조기 형성

### 4.5 네트워크 효과 전략

#### 요양보호사 수 증가 → 보호자 선택지 증가 → 더 많은 보호자 유입 → 더 많은 요양보호사 유입 (선순환)

**네트워크 효과 가속 전략:**

| 단계 | 전략 | 메커니즘 |
|------|------|---------|
| 초기 | 한 지역 집중 | 지역 내 밀도 높이기 → 매칭률 향상 |
| 성장 | 후기 시스템 강화 | 축적된 후기 → 신뢰 형성 → 추가 유입 |
| 성숙 | 케어 데이터 | 더 많은 매칭 데이터 → AI 정확도 향상 → 만족도 상승 |
| 확장 | 양면 커뮤니티 | 요양보호사 커뮤니티 + 보호자 커뮤니티 형성 |

#### 임계 질량 (Critical Mass) 달성 목표

- 지역별 임계 질량: 해당 지역 요양보호사 200명 + 보호자 50명
- 임계 질량 도달 시 자생적 성장 시작

## 5. 해자(Moat) 구축 전략

### 5.1 데이터 해자

**"가장 많은 케어 데이터를 보유한 플랫폼이 최고의 매칭을 제공한다"**

#### 수집 데이터 유형

| 데이터 | 수집 방법 | 활용 |
|--------|---------|------|
| 수급자 케어플랜 | 앱 내 입력 | AI 매칭 알고리즘 |
| 요양보호사 전문성 | 경력, 자격, 후기 | 적합 매칭 |
| 케어 일지/리포트 | 요양보호사 앱 기록 | 품질 관리, 보험 연계 |
| 매칭 결과 | 재이용률, 평점 | 알고리즘 개선 |
| 수급자 건강 변화 | 케어 리포트 내 | 예방적 케어 추천 |

#### 데이터 해자 구축 타임라인

- 6개월: 1만 건 케어 세션 데이터 축적 → 기초 매칭 알고리즘 개선
- 18개월: 10만 건 데이터 → 케어 패턴 분석, 수급자 건강 변화 예측
- 36개월: 100만 건 데이터 → 경쟁사 따라잡기 불가능한 데이터 해자

### 5.2 네트워크 효과 해자

**양면 플랫폼 + 데이터 기반 매칭 = 규모 비례 품질 향상**

핵심 원리:
1. 요양보호사가 많을수록 → 매칭 속도 빠름, 보호자 만족도 상승
2. 매칭 데이터가 많을수록 → AI 정확도 향상 → 더 좋은 매칭
3. 후기가 쌓일수록 → 신뢰도 향상 → 더 많은 보호자 유입

**네트워크 효과 수치화:**
- 특정 지역 요양보호사 100명 → 매칭 성공률 70%
- 특정 지역 요양보호사 500명 → 매칭 성공률 92%
- 매칭 성공률이 높을수록 보호자 재이용률 +15%p

## 8. KPI 및 성과 지표

### 8.1 North Star Metric

> **"월간 활성 매칭 세션 수 (Monthly Active Care Sessions)"**

매칭 건수가 아닌 **실제 케어가 이뤄지는 세션 수**를 핵심 지표로 설정.

이유:
- 매칭 수는 품질과 무관 (1회성 매칭도 포함)
- 활성 케어 세션 = 요양보호사 소득 + 보호자 만족 + 플랫폼 GMV 연동
- 세션이 늘어날수록 데이터 축적 → 알고리즘 개선 → 더 많은 세션 (선순환)

**North Star Metric 목표:**

| 시점 | Monthly Active Care Sessions |
|------|------------------------------|
| 6개월 | 1,000 세션/월 |
| 12개월 | 5,000 세션/월 |
| 24개월 | 20,000 세션/월 |
| 36개월 | 50,000 세션/월 |

## 결론: 왜 바야다인가

### 지금이 최적의 진입 시점인 이유

1. **시장 형성기**: 케어닥이 개념 증명을 완료했지만, 재가요양 전문 디지털 플랫폼 1위는 아직 공백 상태

2. **정책 순풍**: 커뮤니티케어 정책 + 재가요양 급여 확대 → 구조적 수요 증가 불가역

3. **공급 잠재력**: 미활동 요양보호사 195만 명, 이들이 활성화되면 공급 부족 문제 해결 가능

4. **기술 차별화 가능성**: 기존 경쟁사 대부분 기술 수준 낮음, AI 매칭·케어 모니터링으로 즉각 차별화 가능

5. **사회적 가치 정렬**: 요양보호사 처우 개선 + 노인 돌봄 품질 향상 = 사회적 임팩트 + 비즈니스 성공의 일치
